# 📊 easydsd — DART 감사보고서 변환 도구

DART에 제출한 감사보고서 **.dsd 파일을 Excel로 변환**하고,  
당해년도 숫자로 수정한 뒤 **다시 DSD로 복원**하는 도구입니다.

---

## ⬇️ 다운로드 (클릭 한 번으로 설치 없이 실행)

**[👉 최신 버전 다운로드 (Releases 페이지)](../../releases/latest)**

> `easydsd.exe` 하나만 받으면 됩니다.  
> Python, 별도 설치 일절 불필요.

---

## 🖥️ 사용 방법

| 단계 | 설명 |
|------|------|
| 1 | `easydsd.exe` 더블클릭 → 브라우저 자동 실행 |
| 2 | **① DSD → Excel** 탭 → 전년도 `.dsd` 업로드 → Excel 다운로드 |
| 3 | 다운받은 Excel에서 **노란색 셀**을 당해년도 숫자로 수정 |
| 4 | **② Excel → DSD** 탭 → 원본 DSD + 수정 Excel 업로드 → DSD 다운로드 |

---

## ✨ 주요 기능

- 🏦 **재무제표 전체 편집** — 재무상태표, 포괄손익계산서, 자본변동표, 현금흐름표
- 📝 **주석 전체 편집** — 주주명, 지분율, 이자율 등 텍스트 포함
- 🔄 **양방향 변환** — DSD → Excel → DSD, XML 유효성 자동 검증
- 🖱️ **드래그 앤 드롭 UI** — 브라우저 기반, 파일 드래그만으로 사용
- 💡 **자동 종료** — 브라우저 닫으면 서버 자동 종료
